# Random Batch Corpus (50 DOCX)

This folder contains 50 randomly generated DOCX questionnaire files for bulk upload testing.

Design goals:
- Mix of questions already in the current knowledge base (backend/questionnaire_filler.db)
- Mix of intentionally unseen questions that should be flagged
- Multiple layout patterns to stress parser behavior

See `manifest.csv` for expected known vs unknown counts per file.
